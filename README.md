# Getting Started

### Pre-requisites

You need to have `node` and `npm` installed

### Dependencies

To install dependencies, run:
```
npm install
```

This will install all required local dependencies

### Building the project

To build project you can use:

```
npm run build
```

### Running in dev mode:

```
npm run dev
```
